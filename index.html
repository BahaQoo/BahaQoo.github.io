    LINE Messaging API商用範本實作

    利用LINE Messaging API紀錄使用者行為，管理者可以透過後台監控每一位商用帳號好友回應情況、透過後台設定關鍵字來標籤使用者，可以透過標籤分類客戶，並快速推送大量訊息到指定的範圍使用者手機當中。

  • Jun27

    LINE Messaging API 管理者手持介面

    每一則訊息都要重複設定實在是很麻煩，管理者輸入自己驗證好的金鑰碼給API，然後依照規則輸入訊息和指定內容就能發送給使用者們，老實說就是個懶人介面，想要大量洗頻最佳選擇。

  • Jun10

    實作php與asp .net雙平台管理介面

    php要直接把資料庫表格和內容匯出成Excel檔終究沒有像asp .net方便，公司業務又很喜歡將大量的資料全部輸出成Excel來用，果斷做了一個asp的網頁讓php傳值過來呼叫它協助產出檔案。

  • MAY24

    動手實作巴哈的名人語錄

    LINE Messaging API開放讓開發者測試，索性把場外的名人語錄全部收錄下來編列到資料庫裡頭，讓資料在Heroku上頭去運算，然後把使用者回應的訊息習慣紀錄下來，當成BOT開發的實作前哨戰。
